# Events
  - target = el at which event occurs
  - currentTarget = el where event handler attached to

# Arrow functions
  - 'this' refers to context where it is created  

# useContext value
 - pass the setState in separate context for optimizing performance, prevents re-renders
  else, child component will re-render with creation of new value objects

# useEffect
 - no need to specify types explicitly
 - order matters in multiple useEffect calls
